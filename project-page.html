virtual mouse

Description

Part of final semester project.

This project helps the user to control the mouse or on-screen cursor with his hands. This project explores about HCI design and makes room to move from GUI to gesture based systems.This project uses Python,Mediapipe,OpenCV and autopy packages.We used Mediapipe for hand detection.OpenCV captures live camera feed.
